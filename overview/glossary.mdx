This glossary is the canonical short-form reference for terms used across the ABP Framework wiki. Every entry pairs a one-line definition with the file under `framework/src/` that declares it, so coding agents can jump straight to the source. Entries are grouped by concern (modularity, DI, DDD, eventing, infrastructure) but ordered alphabetically inside each group.

## Modularity & Bootstrap

| Term | Definition | Source |
| --- | --- | --- |
| **AbpModule** | Abstract base class every ABP module extends. Hosts `PreConfigureServices`, `ConfigureServices`, `PostConfigureServices`, `OnPreApplicationInitialization`, `OnApplicationInitialization`, `OnPostApplicationInitialization`, `OnApplicationShutdown`. | `framework/src/Volo.Abp.Core/Volo/Abp/Modularity/AbpModule.cs` |
| **DependsOnAttribute** | Class-level attribute that declares transitive module dependencies; implements `IDependedTypesProvider`. | `framework/src/Volo.Abp.Core/Volo/Abp/Modularity/DependsOnAttribute.cs` |
| **IAbpApplication** | Composition root that owns the `IServiceCollection`, `IServiceProvider`, `StartupModuleType`, and provides `ConfigureServicesAsync`/`ShutdownAsync`. | `framework/src/Volo.Abp.Core/Volo/Abp/IAbpApplication.cs` |
| **ServiceConfigurationContext** | Argument passed to `ConfigureServices`; exposes `Services` and a key/value `Items` bag for cross-module wiring. | `framework/src/Volo.Abp.Core/Volo/Abp/Modularity/ServiceConfigurationContext.cs` |

## Dependency Injection

| Term | Definition | Source |
| --- | --- | --- |
| **IConventionalRegistrar** | Pluggable scanner contract with `AddAssembly`/`AddTypes`/`AddType`; ABP registers a default registrar plus per-module ones (e.g. repositories). | `framework/src/Volo.Abp.Core/Volo/Abp/DependencyInjection/IConventionalRegistrar.cs` |
| **ITransientDependency** | Marker interface â€” implementing it makes a class auto-registered with `ServiceLifetime.Transient` by `DefaultConventionalRegistrar`. | `framework/src/Volo.Abp.Core/Volo/Abp/DependencyInjection/ITransientDependency.cs` |
| **IScopedDependency** | Marker interface for `ServiceLifetime.Scoped` auto-registration. | `framework/src/Volo.Abp.Core/Volo/Abp/DependencyInjection/IScopedDependency.cs` |
| **ISingletonDependency** | Marker interface for `ServiceLifetime.Singleton` auto-registration. | `framework/src/Volo.Abp.Core/Volo/Abp/DependencyInjection/ISingletonDependency.cs` |
| **DependencyAttribute** | Overrides lifetime via `Lifetime` and toggles `TryRegister` / `ReplaceServices` semantics during conventional registration. | `framework/src/Volo.Abp.Core/Volo/Abp/DependencyInjection/DependencyAttribute.cs` |
| **ExposeServicesAttribute** | Declares the service contracts a class should be registered as (with `IncludeDefaults`, `IncludeSelf` flags). | `framework/src/Volo.Abp.Core/Volo/Abp/DependencyInjection/ExposeServicesAttribute.cs` |
| **IObjectAccessor\<T\>** | Singleton holder for an object whose value is set after `ServiceCollection` is built; used to bridge build-time and runtime state. | `framework/src/Volo.Abp.Core/Volo/Abp/DependencyInjection/ObjectAccessor.cs` |

## Unit of Work, Filters & Auditing

| Term | Definition | Source |
| --- | --- | --- |
| **IUnitOfWork** | Transactional scope: contains `Items`, `DatabaseApi`/`TransactionApi` registries, `SaveChangesAsync`, `CompleteAsync`, `RollbackAsync`, and `Failed`/`Disposed` events. | `framework/src/Volo.Abp.Uow/Volo/Abp/Uow/IUnitOfWork.cs` |
| **AmbientUnitOfWork** | `IAmbientUnitOfWork` + `IUnitOfWorkAccessor` singleton storing the current UoW in an `AsyncLocal<IUnitOfWork?>`. | `framework/src/Volo.Abp.Uow/Volo/Abp/Uow/AmbientUnitOfWork.cs` |
| **IDataFilter** | Toggle for global query filters such as `ISoftDelete` / `IMultiTenant`; nested `Enable<TFilter>()`/`Disable<TFilter>()` scopes. | `framework/src/Volo.Abp.Data/Volo/Abp/Data/IDataFilter.cs` |
| **IAuditingManager** | Owns the current `IAuditLogScope`; `BeginScope()` returns a save-handle that flushes audit entries when disposed. | `framework/src/Volo.Abp.Auditing/Volo/Abp/Auditing/IAuditingManager.cs` |

## Domain Layer

| Term | Definition | Source |
| --- | --- | --- |
| **IRepository\<TEntity\>** | Persistence-ignorant repository contract; extends `IBasicRepository<TEntity>` + `IReadOnlyRepository<TEntity>` with predicate-based `FindAsync`/`GetAsync` and `IQueryable` access. | `framework/src/Volo.Abp.Ddd.Domain/Volo/Abp/Domain/Repositories/IRepository.cs` |
| **AggregateRoot** | DDD aggregate root base; adds `ExtraProperties` (`IHasExtraProperties`) and `ConcurrencyStamp` (`IHasConcurrencyStamp`) on top of `BasicAggregateRoot`. | `framework/src/Volo.Abp.Ddd.Domain/Volo/Abp/Domain/Entities/AggregateRoot.cs` |
| **BasicAggregateRoot** | Minimal aggregate root: implements `IAggregateRoot` + `IGeneratesDomainEvents`, exposing `AddLocalEvent`/`AddDistributedEvent` and `Get/Clear*Events`. | `framework/src/Volo.Abp.Ddd.Domain/Volo/Abp/Domain/Entities/BasicAggregateRoot.cs` |
| **IDomainService** | Marker interface that auto-registers a service as transient (extends `ITransientDependency`). | `framework/src/Volo.Abp.Ddd.Domain/Volo/Abp/Domain/Services/IDomainService.cs` |
| **ISpecification\<T\>** | Specification-pattern contract with `IsSatisfiedBy(obj)` and `ToExpression()` returning `Expression<Func<T,bool>>`. | `framework/src/Volo.Abp.Specifications/Volo/Abp/Specifications/ISpecification.cs` |
